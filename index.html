Online Piano |

pianoscape.pages.dev

This project allowed me to incorporate both my love for playing the piano and programming by making a piano UI. The design was meant for any beginner to simply open up their phone, tablet, or laptop, and start to learn the groundworks of the instrument. Assisting the user is a Show Key toggle that allows for pressing keys to play notes rather than clicking them. I designed this website using HTMl5 and CSS3 to style the markups of my website, and then implemented the Bootstrap framework for responsiveness across devices. In terms of the back-end, I used JavaScript to show the key command and output notes when they are clicked or key-pressed (or touched on mobile devices).

NBA Player Statistics Engine

This is a Python side project that I made due to my love for the NBA as well as my passion for data analysis. In terms of the interface, I utilized CustomTkinter, which can be described as a prettified Tkinter. Using the BeautifulSoup4 parser library, I was able to scrap the per-game data of individual NBA players, whose names the users can enter into the application, to return the players' image and points, rebounds, and assists a game.
