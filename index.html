NBA Player Statistics Engine

This is a Python side project that I made due to my love for the NBA as well as my passion for data analysis. In terms of the interface, I utilized CustomTkinter, which can be described as a prettified Tkinter. Using the BeautifulSoup4 library, I was able to scrap the per-game data of individual NBA players, whose names the users can enter into the application, to return the players' image and points, rebounds, and assists a game.
